Term GLossary

1. HTML Tag – A tag where all HTML files live within an over-arching html tag. A basic tag that defines an HTML document
2. Syntax – Basic rules of programming in a certain programming language
3. Property – It is what appears before the colon in any line of CSS
4. Browser – A program used to access and display HTML documents
5. Value – is what appears immediately after the colon in any line of CSS
6. Combinator – A character in a selector that connects two selectors together
7. At – Rule – An instruction given in a CSS document using @ character
8. Statement – A statement in CSS is any at-rule or rule set
9. Identifier- is anything that appears as a property, id, class , keyword value, and at-rule
10. Keyword – A value for a property and is somewhat like a reserved word for a particular property
11. Tag – A basic label that defines and separates parts of your markup into elements.
12. Declaration – A set of property names and values
13. Media features – They target the styles depending on the feature of that particular medium
14. Length units – values that declare units in terms of page layout
15. Anchor – A tag used to define a hypertext link
16. Title – A tag that tells the browser what to display as the page title at the top tells search engines what the tittle of your site is
17. Comments – signified by a forward-slash and asterisk, section out an unwanted part of the code
18. Class – part of an element that that names it which you can later style by selecting that certain class
19. ID – elements can have id attributes to identify them, they should always be unique to that single element
20. HREF- links that tell the browser where to go using an HREF attribute, which stores an URL
21. Body – A container that holds all of a page’s content
22. Children – An element that is an immediate descendent of another element, or is nested within another element
23. Div – a block level container or a division of a web page for content with no meaning
24. Head – Tag that surrounds important content that is invisible to the user but is of utmost importance to the browser
25. Headings – Headings are elements ( h1, h2…) that allow you to use six layers of document headings, ranking from largest to smallest.
26. Horizontal line – tag that creates a black line one pixel thick that runs all the way across its container
27. Style sheet – A style sheet includes styling syntax (rules) that dictate how your web page will look
28. Syntax Error – Refers to a situation in which the rules of the computer language are broken or is
29. Frames – HTML supports frames to divide a web page into independent and scrollable sections
30. HTML – stands for Hyper Text Markup Language and is the language used to create all websites
31. Hyperlinks – Links that take the user to another webpage when clicked on
32. Links – Elements that are used to connect your document to another related resource
33. Lists – An unordered or ordered list on the webpage that can be styled in CSS
34. Paragraph – A tag in html that denotes a paragraph of text on the webpage
35. Table – An element for displaying information in rows and columns.
36. Padding – The space between the content and the border ( edge of the element )
37. Margin – The space around the element
38. Font family – A property that sets the font of an HTML element’s text
39. Selector – Used in CSS to select parts of the HTML you want styled
40. Background-color – Attribute that sets the color of a background
41. Font-size – An attribute that sets the size of the font
42. Float – An attribute that makes the whole container of an element push towards a certain side
43. Z-index – An attribute that decides the z-position of an element
44. Margin-top -An attribute that sets the height of the top margin
45. Pixel – A collection of dots that make up a monitor’s display
46. Robot – A software that automatically explores the web
47. Server – A software application that serves requests initiated by client programs
48. Uploading – Moving or sending files to the server
49. Downloading – Moving or retrieving files from the server
50. Navigating – AN act of observing the content of web for some purpose
